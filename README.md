# enterprise-data-engineering-pipeline
End-to-end Apache Spark ETL pipeline that ingests data from REST APIs, MongoDB, PostgreSQL and MinIO, performs data cleansing, transformations and aggregations, stores curated datasets in MinIO and serves analytics-ready data through PostgreSQL.
